 T20I Cricket Match Analysis using SQL

 Overview

This project explores T20 International cricket match data using SQL and SQLite. The analysis focuses on team performance, winning patterns, head-to-head records, match margins, venue statistics, and team rankings.

The project demonstrates the use of advanced SQL techniques to extract meaningful insights from sports data.

Technologies Used

* SQL
* SQLite
* Jupyter Notebook

Dataset

The dataset contains T20 International cricket matches with the following information:

* Team1
* Team2
* Winner
* Margin
* MatchDate
* Ground

Analysis Performed

Team Performance

* Team with the highest number of wins in 2024
* Ranking teams based on total wins
* Win percentage for each team
* Most successful team at each ground

Match Margin Analysis

* Team with the highest average winning margin (runs)
* Team with the highest average winning margin (wickets)
* Matches where the winning margin exceeded the overall average margin

Match Statistics

* Month with the highest number of T20I matches in 2024
* Number of wins while chasing a target (wins by wickets)

Head-to-Head Analysis

* Matches between selected teams
* Head-to-head records between teams
